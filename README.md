## Centralized HTML Dependencies with **htmldeps**

The **htmldeps** package will provide `html_dependency_...()` functions
primarily to support the following packages: **shiny**, **rmarkdown**,
**flexdashboard**, **htmlwidgets**, and **leaflet**.
